# Deret Bilangan Aritmatika

## Deskripsi

Proyek ini bertujuan untuk menghasilkan deret bilangan aritmatika berdasarkan jumlah suku yang diberikan. Deret ini dimulai dari suku pertama 2 dan memiliki beda 3.

## Fitur Utama

- Menghasilkan deret bilangan aritmatika dengan N jumlah suku.
- Memungkinkan pengguna untuk memasukkan jumlah suku yang diinginkan.

## Persyaratan

- Python 3.x

## Instalasi

1. Pastikan Python sudah terinstal di sistem Anda.
2. Clone atau unduh repository ini ke mesin lokal Anda.

   git clone <repository-url>
